This Privacy Notice explains how the Free Open Source IP Leak Test handles your data.

Data Collection

The tool collects the following to provide geolocation and leak testing:

  • IP Address: Used to fetch location details (e.g., city, country) via third-party APIs (ipapi.co, ip-api.com).
  • User Agent: Identifies your browser and device type.
  • WebRTC Data: Checks for IP or device name (mDNS) leaks via WebRTC, if enabled.
  • Geolocation: Uses device coordinates for real-time location on the map, if you enable it.

Data Usage

Your data is used only to:

  • Display IP-based location, ISP, and VPN/Tor status.
  • Detect WebRTC leaks (IP or mDNS identifiers).
  • Show real-time location on the map, if enabled.

Data Storage

No data is stored. All processing is client-side, with API requests sent directly to third-party services.

Third-Party Services

The tool uses:

  • ipapi.co and ip-api.com (ipapi.co, ip-api.com): For IP geolocation. Review their privacy policies.
  • FlagCDN (flagcdn.com): For country flag images.
  • OpenStreetMap (OpenStreetMap) via Leaflet: For maps.

For questions, contact me via GitHub.
