This Privacy Notice explains how the Free Open Source IP Leak Test, developed by sircryptic, collects and uses your data.

Data Collection

The tool collects the following information to provide geolocation services:

  • IP Address: Used to fetch location details (e.g., city, country) via third-party APIs (`ipapi.co` and `ip-api.com`).
  • User Agent: Displayed to identify your browser and device type.
  • Geolocation: If you enable browser geolocation, your device's coordinates are used to show your real location on the map.

Data Usage

Your data is used solely to:

  • Display IP-based location, ISP, and VPN status.
  • Show your real location on the map (with permission).
  • Detect VPN or Tor usage based on ISP or API flags.

Data Storage

No data is stored or retained by the website. All processing occurs in your browser, and API requests are made directly to third-party services.

Third-Party Services

The tool uses:

  • ipapi.co and ip-api.com for IP geolocation data. See their privacy policies for details.
  • flagcdn.com for country flag images.
  • OpenStreetMap via Leaflet for map rendering.

Your Choices

You can:

  • Disable browser geolocation to avoid sharing your real location.
  • Disable JavaScript to prevent IP data collection (note: this will disable the tool's functionality).

Contact

For questions, contact me via GitHub.
